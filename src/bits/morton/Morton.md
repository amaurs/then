# Morton

Part of the color space collection. The Morton code — also known as the Z-order curve — interleaves the binary representations of spatial coordinates to produce a one-dimensional index that preserves two-dimensional locality. This piece applies that interleaving to 24-bit RGB color values, mapping them onto a 4096×4096 canvas where each of the sixteen million colors occupies a unique position.

The Morton code's Z-shaped recursive pattern is visible in the resulting image as a characteristic nested structure of color blocks, each containing smaller blocks in the same arrangement. The piece reveals the Z-order curve's fundamental property: it is a compromise between the one-dimensional ordering that computers require and the two-dimensional adjacency that human perception expects.
