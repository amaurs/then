### Local Development

Add the following to `/etc/hosts` to enable subdomain routing locally:

```
127.0.0.1 blog.localhost calendar.localhost into.localhost poroto.localhost baby.localhost
```

Then run the dev server and access subdomains at:

- `http://blog.localhost:5173`
- `http://calendar.localhost:5173`
- `http://into.localhost:5173`
- `http://poroto.localhost:5173`
- `http://baby.localhost:5173`
- `http://localhost:5173` (main app)
